Create > Light > Directional

==DICE==
Create Empty GameObject > "Dice" > Reset Transform
In Scripts Folder > Create C# Script > "Dice" > Drag to Dice GameObject to create new component
Delete script: Add Using + Class etc

!!IMPORTANT VARIATION!! 
To use TextMeshPro we need to: 
Add > using TMPro;
Replace > Text type variable with TextMeshProUGUI type var
Save and return to Unity

Drag TMPro_Score to new Dice object
